# Clipboard Landing Page

A minimal **Next.js + Tailwind CSS** landing page for Clipboard, showcasing key features and design elements in a clean, responsive layout.

### Key Features

- Hero section highlighting the product.  
- Features section displaying product capabilities.  
- Snippets and reference components for quick visual demonstration.  
- Fully responsive layout with **Tailwind CSS**.  
- Modern design emphasizing clarity and usability.
